# os-dev

## 🇫🇷 Français 

Projet de développement de système d'exploitation qui sera écrit en **Rust** conçu pour approfondir mes compétences en développement bas niveau et démontrer mes capacités techniques.

### ❓ Pourquoi Rust?

Rust est un langage moderne qui devient de plus en plus utilisé. Comme par exemple, dans le Kernel Linux, où Rust est de plus en plus intégré pour sécuriser le code face au C. 
Rust est open-source !
Rust est également très bien supporté sur Arch Linux.

### 💡 Philosophie du Projet

* **100**% humain: Aucune IA n'est ou sera employée dans ce projet. L'objectif étant de montrer des capacités de développement et de réflexion personnelle.
* Low-level de **A-à-Z**: L'objectif étant de comprendre et concevoir nos propres solutions, aucune bibliothèque externe ne sera utilisée dans ce projet.

### ✔️ Objectifs

- [ ] Bootloader personnalisé
- [ ] Initialisation du mode graphique / Display d'image
- [ ] Gestion des interruptions (clavier, horloge système)
- [ ] Gestionnaire de mémoire basique (Pagination / Allocateur)
- [ ] Un mini-shell rudimentaire en ligne de commande

> [!NOTE]
> Plus d'objectif vont venir au fure et à mesure.

## 🇬🇧 English

This project is an operating system written in **Rust**, designed to sharpen my low-level engineering skills and showcase my technical capabilities.

### ❓ Why Rust?
Rust is a modern language gaining significant traction. For instance, it is increasingly integrated into the Linux Kernel to enhance security compared to C. It is also open-source and provides excellent support on my development environment, Arch Linux.

### 💡 Project Philosophy
* **100% Human:** No AI is or will be used in this project. The goal is to demonstrate personal development and problem-solving skills.
* **Low-Level from A to Z:** The focus is on designing my own solutions; no external libraries will be used.

### ✔️ Goals
- [ ] Custom bootloader
- [ ] Graphics mode initialization / Image display
- [ ] Interrupt management (keyboard, system clock)
- [ ] Basic memory management (Paging / Allocator)
- [ ] Rudimentary command-line shell

> [!NOTE]
> More goals will be added as the project progresses.
